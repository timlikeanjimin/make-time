# ⚓ 메이크타임 (웹앱)

> 하루에 단 하나의 하이라이트. **가장 중요한 일을 해낸 하루는 성공입니다.**

책 <메이크 타임>(제이크 냅·존 제라츠키)의 전술을 담은 개인용 시간관리 **PWA**.
아이폰 홈 화면에 앱처럼 추가해서, 잠금화면 위젯 대신 한 번의 탭으로 씁니다.

기존 Scriptable 위젯(`make-time-widget`)을 대체합니다 — 입력·UI가 훨씬 편하고, 오프라인에서도 동작합니다.

---

## 핵심 기능

| 기능 | 설명 |
|---|---|
| ⚓ **오늘의 하이라이트** | 긴급성·만족감·즐거움 기준으로 매일 **딱 하나** |
| ⏰ **마감 시각** | '시각'까지 정하고 남은 시간 카운트다운. 시간이 되면 알림 |
| ✅ **완수 + 색종이** | 완수하면 화면이 초록(항구 도착)으로, 색종이가 터짐 |
| 🔥 **스트릭** | 연속 완수 일수 |
| 📌 **그 외 할 일** | 하이라이트와 섞이지 않게 따로 |
| 🌙 **1분 돌아보기** | 집중·에너지 별점 + 실험일지 (밤 21:30 알림) |
| 📊 **기록** | 최근 7일 점 그래프 + 전체 기록 |

데이터는 기기의 **localStorage**에 저장됩니다(서버 없음, 100% 개인용). 오프라인에서도 켜집니다.

---

## 1. 로컬에서 실행

Node가 설치돼 있으면 (의존성 설치 불필요):

```powershell
cd make-time
node serve.mjs
# → http://localhost:4173
```

또는 그냥 `index.html`을 브라우저로 열어도 동작합니다. (단, 서비스워커·설치 기능은 `http://`로 띄워야 완전히 작동)

---

## 2. 아이폰 홈 화면에 앱으로 추가 (권장 사용법)

웹으로 호스팅한 뒤(아래 3번), 아이폰 **Safari**에서 그 주소를 엽니다.

1. 하단 **공유** 버튼 ⬆︎ 탭
2. **홈 화면에 추가** 선택
3. 추가 → 홈 화면에 ⚓ 아이콘 생성

이제 전체화면 앱처럼 열립니다. (앱 안에서 안내 배너도 떠요)

> 알림(마감·21:30 돌아보기)은 **앱을 홈 화면에 추가**하고 권한을 허용해야 가장 잘 동작합니다. 웹의 한계상, 앱이 완전히 종료된 동안 예약 알림이 항상 보장되진 않습니다 — 앱을 한 번씩 열어주는 흐름을 권장합니다.

---

## 3. 무료로 호스팅 (GitHub Pages)

폰에서 쓰려면 인터넷 주소가 필요합니다. 가장 간단한 무료 방법:

1. github.com 에서 새 저장소 생성 (예: `make-time`, **Public**)
2. 이 폴더의 파일을 업로드 (웹 UI의 *Add file → Upload files* 로 드래그)
   - `gen-icons.mjs`, `serve.mjs`, `README.md`, `.claude/` 는 올리지 않아도 됩니다(앱 동작과 무관).
   - 꼭 올릴 것: `index.html`, `app.js`, `styles.css`, `manifest.webmanifest`, `sw.js`, `icon.svg`, `icon-192.png`, `icon-512.png`, `apple-touch-icon.png`
3. 저장소 **Settings → Pages → Branch: main / root** 저장
4. 1~2분 뒤 `https://<아이디>.github.io/make-time/` 주소가 생김 → 아이폰 Safari로 열어 2번 진행

> 다음 세션에서 "메이크타임 깃허브에 올려줘" 하면 자동 배포를 도와드릴 수 있어요.

## 파일 구조

```
make-time/
├─ index.html              앱 화면
├─ app.js                  로직 (상태·알림·렌더)
├─ styles.css              디자인 (모바일 우선, 상태별 그라데이션)
├─ manifest.webmanifest    PWA 설정
├─ sw.js                   서비스워커 (오프라인 캐시)
├─ icon.svg / *.png        앱 아이콘 (⚓)
├─ gen-icons.mjs           아이콘 PNG 생성기 (재생성용, 배포 불필요)
└─ serve.mjs               로컬 미리보기 서버 (배포 불필요)
```

아이콘을 바꾸려면 `gen-icons.mjs`를 수정 후 `node gen-icons.mjs`.
